Contab

Application SaaS Symfony & React – Gestion Comptable pour Chauffeurs Taxi/VTC (Contab).

🔍 Contexte :
Développement d’une plateforme SaaS dédiée aux chauffeurs indépendants (Taxi/VTC), leur permettant de gérer facilement leur comptabilité, leurs revenus et leurs abonnements, avec une expérience simple et accessible.

⚙️ Solution :
Conception et développement d’une application web moderne basée sur Symfony et React, reposant sur une architecture API-first.

Mise en place d’un système complet incluant :

  • gestion des utilisateurs
  • abonnements (SaaS)
  • intégration avec des services externes

🚀 Réalisations clés :

  • Développement d’une architecture backend robuste avec Symfony & API Platform
  • Mise en place d’une authentification sociale (Google, Facebook, Apple)
  • Implémentation d’un système de gestion des abonnements (SaaS)
  • Création d’APIs pour synchronisation avec services externes
  • Optimisation des performances et de l’expérience utilisateur

🛠️ Stack technique :
Symfony 7 · API Platform · PHP 8 · MySQL · React
