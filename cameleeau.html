CAMELE EAU

Application web dédiée à la gestion de l’autosurveillance du système d’assainissement de la Métropole de Lyon, permettant le suivi, l’analyse et la conformité des données environnementales.

🔍 Contexte :
Intervention sur une application existante nécessitant une modernisation technique complète afin d’assurer sa pérennité, sa compatibilité avec les nouvelles versions des technologies et sa capacité à évoluer.

⚙️ Solution :
Mise à niveau progressive de l’application avec migration vers les versions récentes de Symfony et API Platform, refonte de la structure du projet et correction des incompatibilités liées aux dépréciations PHP.

🚀 Réalisations clés :

  • Migration de Symfony 3 vers Symfony 6
  • Upgrade de API Platform 2 vers API Platform 3
  • Refonte de la structure du projet Symfony (architecture modernisée)
  • Correction des bugs liés aux dépréciations PHP
  • Mise en place et stabilisation de l’application
  • Amélioration de la maintenabilité et des performances globales

🛠️ Stack technique :
Symfony 4 / 5 / 6 · API Platform · Angular · PostgreSQL · PostGIS · Docker
